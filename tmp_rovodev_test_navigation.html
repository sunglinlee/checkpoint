RefreshToken 測試頁面導航

請點擊以下連結測試導航：

方法 1: 直接 URL

在瀏覽器中輸入以下 URL：

http://localhost:3000?page=refresh-token-test

方法 2: JavaScript 導航

在瀏覽器 F12 Console 中執行：

window.location.href = window.location.origin + '?page=refresh-token-test'

方法 3: 檢查當前頁面狀態

在 F12 Console 中檢查：

console.log('Current URL:', window.location.href); console.log('Current page param:', new URLSearchParams(window.location.search).get('page'));
